# 🏏 IPL Data Analysis (2008-2024)
This repository contains a comprehensive Exploratory Data Analysis of Indian Premier League matches. 

## 🚀 Project Overview
In this analysis, I explored the relationships between match outcomes, toss decisions, and venue characteristics using Python. 

### **Key Insights**
* **The Toss Myth:** Winning the toss only gives a **50.5%** edge—making it statistically neutral.
* **The Chasing Trend:** 60% of modern captains choose to **Field First**, highlighting a shift in T20 strategy.
* **Top Performance:** Standardized team names to analyze the true dominance of franchises like CSK and Mumbai Indians.
